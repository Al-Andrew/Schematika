Instructiuni de utilizare:
- Apar in meniul Help din partea de sus a meniului din proram;
!!!Important : trebuie sa apara cate un spatiu intre ficare 'token' al instructiunilor
pe care vreti sa le introduceti in blocuri
Ex: a = a + b (ok) / a=a+b (eroare)

Exista un folder examples, unde sunt o multime de exeple ilustrative pentru folosirea
blocurilor si operatorilor/functiilor disponibile

In fisierul config.hpp se gasesc o multitudine de setari pentru program, cele mai de interes
fiind cele din sectiunea Theme, unde sunt doua teme prestabilite ( alese printr-un #define )

Suntem foarte mandrii de aspectul general al programului si de interfata grafica ( care incearca sa copieze
programele native windows (mai vechi :P))

Aproape fiecare funtie are in headerul unde este declarata o scurta explicatie si autorul(sau autorii)
